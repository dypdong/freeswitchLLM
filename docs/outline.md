# freeswitchLLM 项目文档大纲

> 目标：为 `freeswitchLLM` 建立一套从入门、部署、开发到运维的项目文档结构，帮助团队逐步沉淀 FreeSWITCH 学习与 LLM 集成相关知识。

## 1. 项目概览

### 1.1 项目背景
- 为什么要学习和使用 FreeSWITCH
- FreeSWITCH 在语音通信、呼叫控制、媒体处理中的定位
- 本项目与 LLM 能力结合的设想与边界

### 1.2 项目目标
- 搭建可复现的 FreeSWITCH 学习环境
- 梳理常用 FreeSWITCH 配置与操作流程
- 探索 FreeSWITCH 与 LLM/AI Agent 的集成方式
- 形成面向开发者和运维人员的知识库

### 1.3 适用读者
- 初次接触 FreeSWITCH 的开发者
- 需要调试呼叫流程的后端工程师
- 负责语音平台部署与运维的工程师
- 希望将语音系统与 LLM 应用结合的研发人员

## 5. LLM 集成设计

### 5.1 集成场景
- 语音机器人
- 智能 IVR
- 通话摘要与质检
- 实时转写与意图识别
- Agent 辅助坐席

### 5.2 系统架构
- FreeSWITCH 负责媒体与呼叫控制
- ASR 负责语音转文本
- LLM 负责理解、推理与生成
- TTS 负责文本转语音
- 业务服务负责状态管理与工具调用

### 5.3 集成方式
- Event Socket Library（ESL）事件监听与控制
- MRCP / WebSocket ASR-TTS 接入
- Lua/Python 脚本内嵌控制
- 外部服务通过 API 编排通话流程

### 5.4 数据流与时序
- 呼入流程
- 呼出流程
- 实时对话流程
- 通话结束后的异步分析流程
